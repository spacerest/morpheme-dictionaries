You are creating morpheme dictionary entries for a language-learning word puzzle game. The player knows English and is learning Swahili. For each word provided, create one JSON entry following the guidelines and format below.

**Entry guidelines:**

1. **id**: Use the Swahili word exactly as given (all lowercase).
2. **article**: Always "" — Swahili has no articles. Noun class is encoded in the prefix, not a separate word.
3. **displayPrefix**: Omit — not used for Swahili.
4. **parts**: Split the word at all real morpheme boundaries. Swahili is agglutinative — multiple meaningful prefixes stack before the root, and verb extensions stack after it. Key patterns:

   **Nouns:** The noun class prefix is its own part.
   - m-/mw- (person/plant singular), wa- (people plural), mi- (plants plural)
   - ki-/ch- (thing singular), vi-/vy- (things plural)
   - ji- (object singular), ma- (plural/mass/liquids)
   - u- (abstract), ku- (verbal noun/infinitive)
   - The locative suffix -ni is its own part: nyumba + ni = nyumbani

   **Verb infinitives:** ku- prefix + root (+ extensions): ku + som + a = kusoma
   **Conjugated verbs:** subject marker + tense marker + root (+ extensions + final vowel)
   - ni- (I), u- (you), a- (he/she), tu- (we), m- (you all), wa- (they)
   - -na- (present), -li- (past), -me- (perfect), -ta- (future), -ka- (consecutive)
   - Verb extensions before the final vowel: -ish-/-esh- (causative), -w- (passive), -an- (reciprocal), -ik-/-ek- (stative), -i- (applicative)
   - Final vowel -a is the default; -e for subjunctive; -i for infinitive stem

   The surface form of each part must concatenate exactly to the word's id.
   Aim for 2–4 parts for nouns; 3–5 for conjugated verbs. Add a flag for 6+.

5. **homeLang values**: Each part's homeLang must be SHORT — 1 word ideally, 2 at most. Picture it on a small game tile. Each homeLang is a building block for the player to puzzle out the word — do not pre-solve it. Gloss only what the morpheme itself contributes, not the combined meaning of the compound. Rules: (a) ONE meaning only — no slashes. Pick the single best gloss and put alternatives in homeLangDetails. (b) Noun class prefixes use brief labels: m- = "(person)", wa- = "(people)", ki- = "(thing)", vi- = "(things)", u- = "(abstract)", ku- = "(verb)". (c) Subject markers use the pronoun: ni- = "I", u- = "you", a- = "he/she", tu- = "we". (d) Tense markers: -na- = "(pres)", -li- = "(past)", -me- = "(done)", -ta- = "(fut)". (e) Verb extensions: -ish- = "(cause)", -w- = "(passive)", -an- = "(together)". (f) Roots get one core English word.
6. **translationShort**: A natural, concise English translation (1–4 words).
7. **translationLong**: Longer or alternate translations if useful; otherwise "".
8. **homeLangDetails**: Optional field on a part. For morphemes covered in the glossary you have been given, omit this field — it will be auto-filled. Include it when (a) a morpheme behaves in a context-specific way in this word, or (b) the letters resemble a morpheme but are not functioning as one (e.g., the u- in a verb root is not the abstract noun class prefix). Write in a warm, conversational tone. Never open with "From X..." — write "As in X..." instead (the morpheme is the root; the full form is built on it). Never use em-dashes.
9. **exampleSentence**: A natural, grammatically correct Swahili sentence using the word.
10. **exampleTranslation**: Accurate English translation of the example sentence.

If a word has only 1 morpheme part or 6+ parts, add a "flag" field with a brief note. Otherwise omit "flag".

Return only valid JSON — no explanation, no markdown code fences. Output a single object with a "words" array containing one entry per word.
