**Entry guidelines:**

1. **id**: Use the <TARGETLANG> word exactly as given (lowercase unless language-specific requirements, like uppercase nouns in German, requires it, otherwise no changes).
2. **article**: The definite article for the word if applicable (e.g., "der", "die", "das" for German nouns; "le", "la" for French), or "" for verbs, uninflected forms, and languages without gendered articles. 4. **parts**: Split the word at all real morpheme boundaries, even when a component is itself a recognizable word. For example, "verabschieden" should be ver + ab + schied + en, not ver + abschied, because "abschied" is itself a compound. Aim for 2–4 parts; if full decomposition yields 5 or more, include them all and add a flag. Include linking morphemes (e.g., German -s-) as their own part with "homeLang": "-". Grammatical affixes get a brief label like "(inf)" or "(pl)".
5. **homeLang values**: Each part's homeLang must be SHORT — 1 word ideally, 2 at most. Picture it stamped on a small game tile: the player sees only this. Each homeLang is a building block for the player to puzzle out the word — do not pre-solve it. Gloss only what the morpheme itself contributes, not the combined meaning of the compound. (b) Where a suffix or prefix has a direct English equivalent, use it: -heit → "-ness", -los → "-less", -bar → "-able", -lich → "-like". (c) Context-dependent prefixes get a short parenthetical label: ver- might be "(away)" or "(astray)" depending on the word. (d) Grammatical affixes get brief labels — never spell out long forms: use "(inf)" not "(infinitive)", "(pl)" not "(plural)", "(past)", "(pres)", "(fut)".
6. **translationShort**: A natural, concise translation of the full word (1–4 words).
7. **translationLong**: Longer or alternate translations if useful; otherwise "".
8. **homeLangDetails**: Optional field on a part. For morphemes covered in the core glossary you have been given, omit this field entirely -- it will be filled in automatically. Include it in two situations: (a) when the morpheme does something context-specific in this word beyond its standard glossary meaning (e.g., ver- behaves differently in versprechen vs verabschieden -- that difference is worth a note); or (b) when a part's text resembles a glossary morpheme but is NOT that morpheme in this word (e.g., if the letters "-er" at the end of a word are part of the root rather than the agent suffix, say so). If you do not include homeLangDetails, the standard glossary description will be auto-filled -- so always include it when the standard description would be wrong or misleading. When you do include it, write in a warm, conversational tone, like a knowledgeable friend explaining something interesting, not a dictionary. Never open with "From X..." — write "As in X..." instead (the morpheme is the root; the full form is built on it, not the other way around). Never use em-dashes.
9. **exampleSentence**: A natural, grammatically correct sentence in <TARGETLANG> using the word.
10. **exampleTranslation**: Accurate <HOMELANG> translation of the example sentence.

If a word has only 1 morpheme part or 5+ parts, add a "flag" field with a brief note (e.g., "flag": "only 1 part"). Otherwise omit "flag".
