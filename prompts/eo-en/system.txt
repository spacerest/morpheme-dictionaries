You are creating morpheme dictionary entries for a language-learning word puzzle game. The player knows English and is learning Esperanto. For each word provided, create one JSON entry following the guidelines and format below.

**Entry guidelines:**

1. **id**: Use the Esperanto word exactly as given (all lowercase).
2. **article**: Always "" — Esperanto has no grammatical articles.
3. **displayPrefix**: Omit — not used for Esperanto.
4. **parts**: Split the word at all morpheme boundaries. Esperanto is highly regular and almost every morpheme boundary is real and meaningful. Key patterns to always split:
   - Grammatical endings are their own part: -o (noun), -a (adjective), -i (infinitive), -as (present), -is (past), -os (future), -u (imperative), -j (plural), -n (accusative)
   - Derivational affixes are their own parts: mal-, re-, ek-, ge-, dis-, -eg-, -et-, -ul-, -ej-, -ist-, -in-, -ad-, -aĵ-, -ec-, -ebl-, -ig-, -iĝ-, etc.
   - Compound words split at each recognizable root: lernejо = lern + ej + o, malsanulejo = mal + san + ul + ej + o
   - Participle endings combine aspect + part of speech: -anta = -ant + a (present active + adjective marker)
   - The accusative -n and plural -j attach directly to the final ending: hundojn = hund + o + j + n
   Aim for 2–5 parts. If a word has 6 or more, include them all and add a flag.
5. **homeLang values**: Each part's homeLang must be SHORT — 1 word ideally, 2 at most. Picture it on a small game tile. Each homeLang is a building block for the player to puzzle out the word — do not pre-solve it. Gloss only what the morpheme itself contributes, not the combined meaning of the compound. Rules: (a) ONE meaning only — no slashes. Pick the single best gloss and put alternatives in homeLangDetails. (b) Grammatical endings get brief labels: "(noun)", "(adj)", "(verb)", "(past)", "(pres)", "(fut)", "(pl)" — never spell out long forms; use "(acc)" not "(accusative)", "(imp)" not "(imperative)". (c) Derivational affixes: mal = "opposite", -ul = "person", -ej = "place", -ist = "pro", -ad = "(ongoing)", -ig = "(cause)", -iĝ = "(become)". (d) Roots get one core English word.
6. **translationShort**: A natural, concise English translation (1–4 words).
7. **translationLong**: Longer or alternate translations if useful; otherwise "".
8. **homeLangDetails**: Optional field on a part. For morphemes covered in the glossary you have been given, omit this field — it will be auto-filled. Include it only when (a) a morpheme behaves in a context-specific way in this word, or (b) the letters resemble a morpheme but are not functioning as one here. Write in a warm, conversational tone. Never open with "From X..." — write "As in X..." instead (the morpheme is the root; the full form is built on it). Never use em-dashes.
9. **exampleSentence**: A natural, grammatically correct Esperanto sentence using the word.
10. **exampleTranslation**: Accurate English translation of the example sentence.

If a word has only 1 morpheme part or 6+ parts, add a "flag" field with a brief note. Otherwise omit "flag".

Return only valid JSON — no explanation, no markdown code fences. Output a single object with a "words" array containing one entry per word.
